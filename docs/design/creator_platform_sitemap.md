# Creator Platform Sitemap: GlobalVoice AI

This sitemap outlines the navigational structure of the web-based creator platform.

## 1. Public Facing (Unauthenticated)
- **Home:** Value proposition, hero section, demo video (Before/After AI Dubbing).
- **Features:** Deep dive into Lip-sync, Voice Cloning, and Multi-language support.
- **Pricing:** Tiered subscriptions (Free, Pro, Studio, Enterprise).
- **Resources:** Blog, Case Studies, Help Center.

## 2. Authentication
- **Login:** Email/Social login (Google, Apple).
- **Onboarding:** Creator niche selection, language preferences, and voice sample setup.

## 3. Creator Dashboard (Authenticated)
- **Home / Overview:** 
    - Recent Projects.
    - Usage Stats (Credits/Minutes remaining).
    - "New Project" Quick Action.
- **Project Workspace:**
    - **Step 1: Upload:** Video file upload or URL import (YouTube/Vimeo).
    - **Step 2: Studio (The Core Engine):**
        - Timeline view.
        - Transcript Editor (Source vs. Target).
        - Voice Selection/Cloning toggle.
        - Lip-sync generation toggle.
    - **Step 3: Review & Export:** Preview player with language switching.
- **Voice Library:**
    - My Cloned Voices.
    - Community/Stock Voices.
    - Voice Training Interface.
- **Assets / Media:**
    - Uploaded source videos.
    - Exported localized assets.
- **Settings:**
    - Profile Management.
    - API Keys (for developers).
    - Billing & Invoices.
    - Team/Workspace Permissions.

## 4. Support & Feedback
- **In-app Chat:** Live technical support.
- **Feedback Loop:** Feature requests and bug reporting for the AI models.
